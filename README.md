# 01 : Image Guidelines

### 🖼️ Recommended Image Asset Guidelines

To maintain high performance and visual consistency across the **Al-Hilal Panjabi** platform, follow these image size recommendations. These are optimized for Cloudinary delivery and Next.js performance.

| Image Type | Recommended Size | Notes |
| :--- | :--- | :--- |
| **Product Images** | 1000x1000px | Square format, high quality for zoom feature. |
| **Product Thumbnails** | 300x300px | Optimized for grid views and fast loading. |
| **Hero Slider Images** | 1920x1080px (16:9) | Full-width banner, Cloudinary optimization applies. |
| **Banner Images** | 1600x500px | Horizontal banners for homepage sections. |
| **Promo Banners** | 800x500px | Smaller promotional cards for offers. |
| **Category Banners** | 1200x600px | Category page headers. |
| **Logo** | 300x90px | Must be Transparent PNG or SVG. |
| **Favicon** | 32x32px | ICO or PNG format. |

---

### ⚡ Optimization Tips:
* **Format:** Use `.webp` for general images and `.svg` for icons/logos to ensure fast loading.
* **Hero Slider:** Ensure the focal point of the image is centered to avoid clipping on mobile devices.
* **Cloudinary:** All dynamic images should be uploaded via the Admin Panel to leverage auto-optimization.
